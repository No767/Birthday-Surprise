## Install + Build

Requirements: 

- pnpm
- node.js

Install:

```sh
pnpm install
```

Running dev server:

```sh
pnpm run dev
```

Build for prod (u dont have to do this since this site deploys to netlify):

```sh
pnpm run build
pnpm run start
```
